# Agent-
什么是Agent，llm+tools。
Agent流该如何创建？
首先明确什么是agent，我认为agent=llm+tools，大模型长出血肉（
1.大模型 Brain
这是agent的大脑，所以一切行为基于此。这里可以直接调用api，比如deepseek，非常优秀的且亲民的大模型。
2.工具 Tools
这里相当于一个转折点了，25年开始兴起，知道现在如何让大模型合理且快速的调用tools是现在很现实的一个问题，我把这个阶段称为FAGI，Fake AGI，对于大部分人现在的agent能做到太多不可思议的事情了，几分钟写出千行代码并且正确跑通，vibe coding的大火，这一切都是从agent的出现开始的。
3.记忆 Memory
这里对于agent是更加智能化的体现，拥有memory的agent是可以一句话做到很多事情的，比如code review和一句简单的：开始跑吧老伙计！他就可以开始工作。
4.规划 Planning 
一个Work flow，比如一步一步该要做什么，我要去吃饭，我先穿鞋拿手机，然后走哪条路到饭店等等。
